  🧠 HealthPlus – Smart Health Companion
HealthPlus is a full-stack health tracking platform with an integrated AI assistant, designed to help users monitor, analyze, and improve their overall well-being.

   
   ⚙️ Tech Stack

* **Frontend:** Next.js, React, Tailwind CSS
* **Backend & DB:** Firebase Authentication, Firestore
* **AI:** Ollama (LLaMA3:8B) – local LLM

   🤖 HealthBot (Core Feature)

* AI-powered chatbot with streaming responses
* Context-aware replies using user health data
* Chat history with persistent storage
* Designed to act as a personal health assistant



  🧩 Features

    1. Nutritional Wellbeing

* Food logging & calorie tracking
* Nutrition calendar visualization
* Recipe suggestions
* Food insights dashboard

   2. 😴 Physical Wellbeing

* Sleep tracking & analytics
* Sleep calendar view
* BMI calculator
* Water intake tracking

    3. Mental Wellbeing

* Journaling system (Dear Diary)
* Mood tracking via calendar
* Self-assessment tests

    4. Social / Team Features

* Team dashboard with shared stats
* Leaderboards (sleep & calories)
* Comparative analytics across users




  🔮 Future Scope

* AI-based health insights & predictions
* Mood detection using NLP
* Food image recognition
* Personalized health scoring
